Simulation of Hollow-Fiber Membrane Module for Point Source Carbon Capture

Graduate Research Assistant with Prof. Miao-Jung Yvonne Ou (Advisor) & Dr. Robert DeJaco (Ardent Process Technologies)

Overview

This project develops a mathematical model of a hollow-fiber membrane module designed for point-source carbon capture. The system separates CO₂ from N₂ by exploiting selective permeability across a membrane separating concentric lumen and shell regions. The governing equations are discretized and implemented in FEniCS.

Collaborative · 2026

SUM Innovation — Decentralized Infrastructure for AGI

Collaborator SUM Chain · OmniNode Protocol

The Problem

Three hyperscalers control 85%+ of global AI compute. Running a 70B model requires $30K+ in GPU hardware that sits idle 90% of the time. Every inference request sends private data to corporate servers — creating cost barriers, data sovereignty risks, and single points of failure.

Two Interconnected Systems

⬡

OmniNode Protocol

A peer-to-peer AI inference network written in 100% Rust that distributes LLM execution across consumer hardware using pipeline parallelism.

  • Model layers sharded across devices over a P2P mesh
  • Federated learning — only gradients leave the device, never raw data
  • zkML proofs verify correct inference
Ϙ

SUM Chain (L1 Blockchain)

A fixed-supply, immediate-finality blockchain underpinning the network economy. Guiding principle: "do not trust us; trust math."

  • Native currency Koppa (Ϙ) with fixed supply and deflationary fee burn
  • Tendermint-style BFT consensus with immediate finality
  • Native NFTs for certified document tokenization — no smart contracts needed
